Academic Projects

These are the group projects that I took part in throughout my bachelor's journey:

  • MULTICLASS BRAIN TUMOR CLASSIFICATION - A Machine learning based thesis project submitted in 8th semester. In this project, a Convolutional Neural Network (CNN) model was implemented to detect brain tumors from MRI Images.
  • RUMOR DETECTION - A Machine learning based thesis project submitted in 7th semester. It was a Natural Language Processing (NLP) challenge to detect rumorous and non-rumorous text.
  • INTEL IMAGE CLASSIFICATION - A Machine learning based image classification challenge where a deep learning model was implemented to classify six types of images. Visit Kaggle.
  • CUSTOMER CHURN PREDICTION - A Data Science and Machine Learning based project where important insights were extracted from customer feedback to predict if in future the customer might churn or not.
  • FOOD ORDERING WEB APPLICATION - A web application written in HTML, CSS, JavaScript, and PHP. In this web application, people can order food online as well as a vendor can manage his business.
  • ONLINE GROCERY SYSTEM - It is a complete e-commerce web application written in HTML, CSS, JavaScript, and PHP where people can order grocery items online, vendors can manage their businesses, and an admin/owner can handle the whole system.
  • MY DOCTOR APPLICATION - It is an android application where the list of all doctors, hospitals, clinics, and medical emergency contacts in my locality were collected and created a mobile application. The motivation was to help my community.
  • RAIL HELPER - It is an android application. Trains are always late in my country. So we were motivated to build such a platform where people can share some location specific train's position. The main motto was sharing is caring - the people/passenger will share information and the people themselves will be benefitted.
